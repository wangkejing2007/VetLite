# VetLite - 獸醫學入門互動學習系統 🐾

**VetLite** 是一款專為獸醫學生與愛好者設計的輕量級互動學習軟體。透過直觀的介面與遊戲化的體驗，讓艱澀的醫學知識變得觸手可及。

## 🌟 主要功能 (Features)

### 1. 📚 知識卡片 (Flashcards)
- 內建基礎解剖與生理學題庫。
- **[新增功能]** 支援使用者自定義卡片，可新增個人筆記，資料會永久保存在瀏覽器中。

### 2. 🦴 互動解剖室 (Interactive Anatomy)
- 支援 **5 大物種**：狗 🐶、貓 🐱、鳥 🦜、馬 🐎、魚 🐠。
- 每個物種擁有 **10 個關鍵解剖熱點**。
- 點擊熱點即可查看詳細的臨床意義說明。

### 3. 🕵️‍♀️ 臨床小偵探 (Clinical Detective)
- 扮演值班獸醫，挑戰真實臨床病例。
- 收錄 **10 個經典教案**，包含巧克力中毒、貓導尿、鸚鵡卡蛋等急診情境。
- 即時回饋系統，解析正確與錯誤的決策原因。

## 🚀 如何使用 (How to Run)

本專案採用純靜態網頁技術 (HTML5 / CSS3 / Vanilla JS)，無需安裝任何伺服器或環境。

1. 下載或解壓縮本專案資料夾。
2. 在資料夾中找到 `index.html`。
3. 雙擊檔案，使用任何現代瀏覽器 (Chrome, Edge, Safari) 開啟即可。

## 🛠️ 技術架構
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Storage**: LocalStorage (用於儲存使用者筆記)
- **Design**: Mobile-First, Responsive Design (PWA Ready)
- **Assets**: 離線運行的輕量化圖片資源
